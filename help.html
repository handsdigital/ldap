LDAP Connect - Conta Mínima

Este documento visa auxiliar o administrador de softwares Conta Mínima implantar o login integrado ao LDAP.

Configurações necessárias

Para perfeito funcionamento deste integrador de LOGIN, é necessário informar no arquivo config.php as informações referentes a seu LDAP.
O usuário informado neste arquivo deve ter permissão de listar todos os usuários na base do LDAP.
Caso seja alterado a senha deste usuário no LDAP, devera ser alterado neste arquivo.
Deve-se informar os campos no LDAP que representam estas informações (no cadastro do colaborador):

  • Nome do Colaborador;
  • E-Mail do Colaborador;
  • Nome do Usuário do Colaborador;
  • Grupo do Colaborador.
